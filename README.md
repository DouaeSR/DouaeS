# BookShop API

REST API for managing an online bookstore built with Spring Boot, MySQL, Docker, and JWT authentication.

---

## Project Information

- Framework: Spring Boot
- Database: MySQL
- Authentication: JWT
- Containerization: Docker
- CI/CD: GitHub Actions
- Deployment: Linux Server
